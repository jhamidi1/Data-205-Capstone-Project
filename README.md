# Neighborhood Socioeconomic Analysis - By Jathiya Hamidi

## Data 205 Capstone Project

## Overview
Do differences in education really shape income and poverty across communities?

This project will be exploring the relationship between educational attainment, income levels, and poverty rates across neighborhoods in Pittsburgh. By combining multiple datasets using neighborhood as a common link, the goal is to identify patterns and better understand how education connects to socioeconomic outcomes.

---

## Data Sources
The datasets used in this project come from publicly available Pittsburgh neighborhood data. Each dataset includes estimates for different socioeconomic factors.

Main variables used:
- Educational attainment
- Income levels
- Poverty rates
